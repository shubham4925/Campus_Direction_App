# Campus_Direction_App
The Campus Direction App is a mobile application designed to help students, visitors, and staff easily navigate around a college or university campus. The app provides directions to classrooms, laboratories, offices, libraries, cafeterias, parking areas, and other important locations within the campus

# Features

📍 Interactive campus navigation

🗺️ Easy route guidance between locations

🏫 Search for buildings, departments, and facilities

📌 Real-time location tracking

🔍 Quick location search

📱 User-friendly interface

⚡ Fast and accurate directions


# Technologies Used

Android Studio

Java/Kotlin

Google Maps API

Firebase (Optional)

XML for UI Design


# Objective

The main objective of this project is to simplify campus navigation and reduce the time required to locate buildings and facilities, especially for new students and visitors.

Future Enhancements

Indoor navigation support

Voice-guided directions

# Conclusion

The Campus Direction App improves accessibility and navigation within educational institutions by providing an efficient and user-friendly solution for finding campus locations
